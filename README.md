## Instructions

To run application:
```
npm start
```

To see page with status bar:
> http://localhost:8080/webpack-dev-server/index.html

Packages:
```
npm i bootstrap -s
npm i -s react-router
npm i -s react-router-dom
npm i -s history
npm i -s redux
```
